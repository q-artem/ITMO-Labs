Способы запуска

  1. Для запуска с использованием Java Web Start необходимо перейти по ссылке. Данный способ гарантирует использование актуальной версии эмулятора БЭВМ и рекомендуется для удалённого использования.
  2. Для запуска в качесте Java Applet из командной строки необходимо выполнить команду: 
    appletviewer http://helios.cs.ifmo.ru/bcomp/bcomp.html
    На компьютере должна быть установлена полная версия JDK.
  3. Для запуска в качесте Java Applet из Web-браузера необходимо перейти по ссылке. После запуска приложения необходимо нажать на клавишный регистр для захвата приложением фокуса.
  4. Зайти на сервер кафедры ВТ helios и выполнить команду: 
    bcomp
    Данный способ рекомендуется при работе в аудитории 375. Удалённый запуск с сервера helios возможен, но требует наличия на клиенте X сервера и умения организации проброса X11.
  5. Скачать JAR-архив и запустить эмулятор командой:
    6. java -jar bcomp.jar
  6. Для unix-подобных систем дополнительно скачать shell скрипт и запустить его командой:
    7. bcomp
  7. Для ОС семейства Windows скачать JAR-архив и запустить его двойным нажатием левой клавиши мыши.
  8. Для ОС семейства Windows дополнительно скачать bat скрипт и запустить его командой:
    9. gui

Микропрограммы

  1. Исходная микропрограмма используется по умолчанию.
  2. Оптимизированная микропрограмма используется при указании ключа -Dmp=optimized. При использовании shell скрипта bcomp необходимо указать ключ -o. Использование оптимизированной микропрограммы невозможно при запуске эмулятора через Java Web Start или в качестве Applet.
  3. Расширенная микропрограмма используется при указании ключа -Dmp=extended. При использовании shell скрипта bcomp необходимо указать ключ -e. Использование расширенной микропрограммы невозможно при запуске эмулятора через Java Web Start или в качестве Applet.

Ограничения и известные проблемы

  1. Графический интерфейс недорисован.
  2. При ручном изменении микропрограммы возможно неправильное определение выполняемого цикла.
