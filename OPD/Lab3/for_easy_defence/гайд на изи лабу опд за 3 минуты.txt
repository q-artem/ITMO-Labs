Итак гайд на изи лабу по ОПД за 3 минуты, погнали
1) Пропарсить текст с картинки можно прилагающейся простой утилкой написанной на РАСТПОБЕДА (спасибо ивабусу)
Заходите в консоль (если на винде, то открываем cmd в директории с файлом). Вписываете туда

curl --data-binary "@test4.png" https://bcomp-ocr.ivabus.dev/commands

Где вместо test4.png название вашей картинки с вашими командами скачанный с se.ifmo.ru. Так мы сразу получаем ваши команды для автоматического создания трассировки.
2) Далее загружаем наш код на трассировку (её делаем через консольный режим). Больше можно узнать в cli.html

echo"
{your initial address} a
0200 w
4501 w
{command} w
...

{address where program starts} a

c c c c c c c c c c ... {as much as needed}
" | java -jar -Dmode=cli ../bcomp-ng.jar

3) Теперь мы уже готовы показывать нашу прогу преподователю, нужно как-то загрузить ёё в БЭВМ.
Делаем самым тупым образом через ассемблер (больше об ассемблере в asm.html). Заходим на вкладку ассемблер, вставляем, жмякаем "компилировать".

ORG 0x239 {here you put your initial address}
{after "WORD" you put all of your commands}
BEGIN: WORD 0x024E, 0xA000, 0x4000, 0x0200, 0x0200, ...

Теперь ставим в БЭВМ адрес начала вашей программы, не забываем нажать F9 и F7 после него.
4) Вам выдали доп, советую писать его в shell скрипте и запускать через тот же скрипт. Как видно, в скрипте мы можем комментировать строчки (практик скажет спасибо) и консольный режим БЭВМ покажет Вам значение всех переменных которые вы использовали (благодаря ru s c).

echo "
239 a
024D w
0019 w
0200 w # старт (23B)
AAFC w # автоинкрементная проходка по массиву из 24D (JUMP был сюда)
...

23B a

sl 0
ru
s c


" | java -jar -Dmode=cli ../bcomp-ng.jar
