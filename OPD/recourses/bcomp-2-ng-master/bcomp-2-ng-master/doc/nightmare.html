Способы запуска

  1. Зайти на сервер кафедры ВТ helios и выполнить команду: 
    bcomp -n
    Данный способ рекомендуется при работе в аудитории 375. Удалённый запуск с сервера helios возможен, но требует наличия на клиенте X сервера и умения организации проброса X11.
  2. Скачать JAR-архив и запустить эмулятор командой:
    3. java -Dmode=nightmare -jar bcomp.jar
  3. Для unix-подобных систем дополнительно скачать shell скрипт и запустить его командой:
    4. bcomp -n

Микропрограммы

  1. Исходная микропрограмма используется по умолчанию.
  2. Оптимизированная микропрограмма используется при указании ключа -Dmp=optimized. При использовании shell скрипта bcomp необходимо указать ключ -o.
  3. Расширенная микропрограмма используется при указании ключа -Dmp=extended. При использовании shell скрипта bcomp необходимо указать ключ -e.

Ограничения и известные проблемы

  1. Взаимодействие с микропрограммным устройством управления не предусмотрено.
  2. Возможность работы с ВУ9 не реализована.
